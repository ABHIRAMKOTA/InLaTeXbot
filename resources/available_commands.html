General commands /start - display the greeting /help - display this help /abort - stop any ongoing operation, i.e. preamble customization Preamble control /setcustompreamble - set up a custom preamble that will be used to generate images /getmypreamble - see the current preamble registered for your user id /getdefaultpreamble - see the default preamble For more help please see the project's page on Github: https://github.com/vdrhtc/InLaTeXbot
