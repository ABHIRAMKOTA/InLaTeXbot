General commands /start - display the greeting /help - display this help /abort - stop any ongoing operation, i.e. preamble customization Preamble control /registerCustomPreamble - set up a custom preamble that will be used to generate images /getMyPreamble - check the current preamble registered for your user id /getDefaultPreamble - see the default preamble For more help please see the project's page on Github: https://github.com/vdrhtc/InLaTeXbot
